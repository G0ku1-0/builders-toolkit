# 🧰 Builder's Toolkit

> **Build smarter. Stay vanilla.**

Builder's Toolkit is a vanilla-friendly Minecraft Bedrock Edition resource pack designed for builders, technical players, and redstone engineers. It enhances visual clarity while preserving the vanilla look and remaining achievement-friendly.

## ✨ Features

### Current
- Vanilla-style design
- Lightweight resource pack
- Achievement-friendly

### Planned
- 📦 Hopper direction indicators
- 👀 Observer direction indicators
- 🧲 Sticky piston indicator
- ⚙️ Dropper & dispenser differentiation
- 📐 Chunk border visualization
- 🧍 Armor stand technical markers
- 🐚 Nautilus Shell builder mode
- 🧭 Direction & orientation helpers
- 🏗️ Building alignment guides

## 🎯 Goals

- Preserve Minecraft's vanilla aesthetic.
- Improve building and redstone workflows.
- Stay lightweight and performance-friendly.
- Keep achievements enabled.

## 📥 Installation

1. Download the latest `.mcpack` from the Releases page.
2. Open the file with Minecraft.
3. Activate the resource pack in your world or Global Resources.
4. Enjoy building!

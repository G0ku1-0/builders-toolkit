# Contributing to Builder's Toolkit

Thank you for your interest in contributing to Builder's Toolkit!

## How to Contribute

### 🐞 Report Bugs

Open a GitHub Issue with:
- Minecraft version
- Device/Platform
- Description of the issue
- Screenshots (if applicable)

### 💡 Suggest Features

Before implementing a feature, please open an Issue describing:
- What the feature does
- Why it would be useful
- Any reference images or examples

### 🔧 Submit Changes

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test them in Minecraft Bedrock Edition.
5. Submit a Pull Request.

## Guidelines

- Keep the vanilla Minecraft aesthetic.
- Avoid unnecessary visual clutter.
- Maintain compatibility with achievements.
- Keep textures clean and readable.
- Use descriptive commit messages.

## Pull Requests

Please explain:
- What changed
- Why it changed
- Any compatibility concerns

The project owner reviews all pull requests and may request changes before merging.

## Community Guidelines

Be respectful and constructive.

Harassment, discrimination, spam, or abusive behavior will not be tolerated.
